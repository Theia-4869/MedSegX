# MedSegX
This is the official repository for A generalist foundation model and database for open-world medical image segmentation (MedSegX).

## Train
Training:
```
python pretrain.py
```
Fine-tuning:
```
python finetune.py
```

## Evaluate
Internal evaluation:
```
python evaluate_internal.py
```
External evaluation:
```
python evaluate_external.py
```
