# MedSegX
This is the official repository for A generalist foundation model and database for open-world medical image segmentation (MedSegX).

## Train
Training:
```
python pretrain.py
```
Fine-tuning:
```
python finetune.py
```

## Evaluate
ID evaluation:
```
python evaluate_id.py
```
OOD evaluation:
```
python evaluate_ood.py
```
